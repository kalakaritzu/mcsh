Install

tar -xzf mcsh-linux-x64.tar.gz
chmod +x mcsh
sudo mv mcsh /usr/local/bin/mcsh

What's new

  • First Linux release
  • Quick CLI launch — mcsh run <name> from any terminal
  • Modpack update — modpack update checks all packs for newer Modrinth versions
  • Version picker on install — grouped by MC version with loader and age
  • Session persistence — active instance remembered across restarts
  • Java auto-install downloads the correct Linux JDK from Adoptium
  • Instance/mod folder open uses xdg-open
  • All other v0.3.4 features and bug fixes
